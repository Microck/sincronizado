description: Fast fixes for common failures.
---

## Sync slow

```bash
mutagen sync list
```

Check ignores: `.opencode.config.json` -> `sync.ignore`.

## ET disconnects

```bash
sudo systemctl restart et
```

## Agent-OS 502

Agent not running or crashed.

```bash
sudo systemctl status agent-os
sudo journalctl -u agent-os -f
```

## Port 3000 blocked

```bash
sudo ufw allow 3000/tcp
sudo ufw status
```
