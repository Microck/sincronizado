# Sincronizado NPM Package

## Installation

```bash
npm install -g sincronizado
```

## Usage

```bash
# Start development session
sincronizado

# List sessions
sincronizado --list

# Create config
sincronizado-config
```

## Configuration

Create `.opencode.config.json` in your project root.

See full documentation at: https://microck.github.io/sincronizado/
