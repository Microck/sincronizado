# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   | :white_check_mark: |
| < 1.0   | :x:                |

## Reporting a Vulnerability

If you discover a security vulnerability, please report it responsibly:

1. **DO NOT** open a public issue
2. Email security concerns to: [microck@example.com] (replace with actual email)
3. Include detailed reproduction steps
4. Allow time for assessment before public disclosure

We will:
- Acknowledge receipt within 48 hours
- Provide regular updates on remediation
- Credit you in any public disclosure (with your permission)

## Security Best Practices

When using Sincronizado:

- Keep Tailscale up to date
- Use strong SSH keys (ed25519)
- Rotate credentials regularly
- Monitor access logs on your VPS
- Keep OpenCode and plugins updated
