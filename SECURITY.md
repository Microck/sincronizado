# Security Policy

## Supported Versions

We currently support the latest released version on GitHub.

## Reporting a Vulnerability

If you discover a security vulnerability, please report it responsibly:

1. **DO NOT** open a public issue
2. Report it via GitHub Security Advisories:
   https://github.com/Microck/sincronizado/security/advisories/new
3. Include detailed reproduction steps and impact
4. Allow time for assessment before public disclosure

We will:
- Acknowledge receipt within 48 hours
- Provide regular updates on remediation
- Credit you in any public disclosure (with your permission)

## Security Best Practices

When using Sincronizado:

- Keep Tailscale up to date
- Use strong SSH keys (ed25519)
- Rotate credentials regularly
- Monitor access logs on your VPS
- Keep OpenCode and plugins updated
