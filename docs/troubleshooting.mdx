description: Fast fixes for common failures.
---

## Sync slow

```bash
mutagen sync list
```

Check ignores: `~/.config/sincronizado/config.json` -> `sync.ignore`, plus any project `.syncignore`.

## Mutagen not found

If `sinc` prints "Mutagen is required for file sync":

- run `sinc --setup` to install it automatically
- or install manually from https://mutagen.io/

## Session already exists

If you see "Session ... already exists":

```bash
sinc -r
```

## Can't connect to VPS

1. Verify SSH works:

```bash
ssh user@hostname "echo ok"
```

2. If you configured an SSH key in the setup wizard, ensure the path exists and permissions are correct.

## Disable update checks

```bash
SINC_NO_UPDATE_CHECK=1 sinc
```
