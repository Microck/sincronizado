Get up and running with sincronizado in 5 minutes.

## Prereqs

- Windows 10+ or macOS 12+
- VPS: Ubuntu 20.04+ or Debian 11+
- Tailscale account
- Git

## Install

### Option 1: One-liner install (recommended)

Installs the `sinc` CLI.

Linux / macOS:

```bash
curl -fsSL https://sync.micr.dev/install.sh | bash
```

Windows (PowerShell):

```powershell
irm https://sync.micr.dev/install.ps1 | iex
```

Then run the setup wizard:

```bash
sinc --setup
```

This writes your config to `~/.config/sincronizado/config.json` and verifies SSH + Mutagen.

### Option 3: Manual setup

### 1) deps

Windows (PowerShell):

```powershell
winget install Mutagen.Mutagen
```

macOS:

```bash
brew install mutagen
```

### 2) clone

```bash
git clone https://github.com/Microck/sincronizado.git
cd sincronizado
```

### 3) vps setup

```bash
# Standard mode with OpenCode (default)
sudo ./scripts/setup-vps.sh

# With Claude Code instead
sudo ./scripts/setup-vps.sh --agent=claude

# Or with specific components
sudo ./scripts/setup-vps.sh --mode=custom --with-kimaki --with-lunaroute
```

### 4) config

```json
{
  "sync": {
    "mode": "both",
    "ignore": ["node_modules", ".venv", ".git"],
    "remoteBase": "~/workspace"
  },
  "vps": {
    "hostname": "your-vps.tailnet.ts.net",
    "user": "ubuntu",
    "port": 22
  },
  "agent": "opencode"
}
```

Save this as `~/.config/sincronizado/config.json`.

### 5) start

From your project directory:

```bash
sinc
```

## Verify

1. Check sync: edit a file locally, verify it appears on VPS in &lt;500ms
2. Test resilience: switch networks, session stays alive
3. Mobile access: open `http://your-vps:3000`
4. Optional (Discord): if you installed Kimaki, verify bot is online

## Installation modes

| mode     | components                            | use case                |
| -------- | ------------------------------------- | ----------------------- |
| minimal  | ET, OpenCode/Claude, UFW              | headless servers, CI/CD |
| standard | + Agent-OS, ccmanager, plugins        | recommended             |
| full     | + Kimaki, LunaRoute, worktree-session | power users             |
| custom   | pick components                       | specific needs          |

### Optional components

TUI installer: pick these in the UI.

Manual VPS setup: `./scripts/setup-vps.sh` supports flags:

```bash
--agent=claude             # use Claude Code instead of OpenCode
--with-kimaki              # discord bot for voice/text control
--with-lunaroute           # ai proxy with token tracking
--with-worktree-session    # git worktree per session
--with-session-handoff     # context continuation
--with-agent-of-empires    # alternative session manager

--no-agent-os              # skip web ui
--no-ccmanager             # skip session manager
--no-plugins               # skip opencode plugins
```
