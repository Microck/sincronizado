Get up and running with `sinc` in a few minutes.

## Prereqs

Local:

- SSH client
- Git

VPS:

- Linux (Ubuntu/Debian recommended)
- SSH access (key-based recommended)

## Install

Linux / macOS:

```bash
curl -fsSL https://sync.micr.dev/install.sh | bash
```

Windows (PowerShell):

```powershell
irm https://sync.micr.dev/install.ps1 | iex
```

## Setup

Run the setup wizard once:

```bash
sinc --setup
```

It writes `~/.config/sincronizado/config.json`, verifies SSH, installs local dependencies (Bun + Mutagen) if missing, and can run VPS bootstrap.

## Connect

From your project directory:

```bash
sinc
```

## Manage sessions

```bash
sinc --list
sinc --kill <id>
```

If you disconnect and want to resume:

```bash
sinc -r
```

## Verify

- Sync sessions:

```bash
mutagen sync list
```

- VPS sessions:

```bash
ssh user@hostname "tmux ls || true"
```
