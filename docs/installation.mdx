title: Installation
description: Install sincronizado and run the setup wizard.
---

Install `sinc`, then run the setup wizard once.

## One-liner install

macOS / Linux:

```bash
curl -fsSL https://sync.micr.dev/install.sh | bash
```

Windows (PowerShell):

```powershell
irm https://sync.micr.dev/install.ps1 | iex
```

## Setup

```bash
sinc --setup
```

The wizard:

- collects VPS details (hostname/user/port)
- configures SSH key usage (use existing or generate new)
- installs local dependencies (Bun + Mutagen) if missing
- writes `~/.config/sincronizado/config.json`
- optionally runs the VPS setup script (`./scripts/setup-vps.sh`) over SSH

## Verify

```bash
sinc --version
```

From a project directory:

```bash
sinc
```

## Uninstall

```bash
sinc --uninstall
```

This removes local config and terminates the sync session for the current project.
