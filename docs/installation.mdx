Install `sinc`, then run the setup wizard once.

## One-liner install

macOS / Linux:

```bash
curl -fsSL https://sync.micr.dev/install.sh | bash
```

Windows (PowerShell):

```powershell
irm https://sync.micr.dev/install.ps1 | iex
```

## Setup

```bash
sinc --setup
```

The wizard collects VPS details, tests the connection, installs dependencies, and writes your config file.

## Verify

```bash
sinc --version
```
