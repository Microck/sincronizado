# Scripts

This directory contains setup and automation scripts for Sincronizado.

## Structure

- `setup-vps.sh` - VPS setup script (Ubuntu/Debian)
- `setup.ps1` - Windows local setup
- `setup.sh` - macOS local setup
- `validate.ps1` / `validate.sh` - Configuration validation
